## Cloud 9 Notes

Cloud 9 supports:

- C++ 11
- C++ 14

The C++ Workspace provides:

- GCC version 4.8.4

To use C++ 11 functions pass

```
-std=c++11

/usr/bin/g++ -std=c++11 $file -o runme; chmod 755 $file_path/runme; $file_path/runme
```

To use C++ 14

```
/usr/bin/g++-5 -std=c++14 $file -o runme; chmod 755 $file_path/runme; $file_path/runme
```
